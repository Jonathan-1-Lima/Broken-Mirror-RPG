# Broken-Mirror-RPG
Broken Mirror RPG é um projeto de RPG de mesa digital em 2D, inspirado em jogos como Gather Town.  O projeto possui movimentação livre, múltiplas salas, sistema de interação com objetos, mesa de RPG com assentos, mestre da mesa, controle de turnos e rolagem de dado D20 animada.  Sistema é (single player), com planos futuros para multiplayer online.
